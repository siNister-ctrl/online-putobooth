# Couple Booth 💕

A real-time collaborative online photobooth for couples. Capture memories together with synchronized layouts, filters, stickers, and more.

## Deploy to Vercel

1. Push this project to a GitHub repo
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Vercel auto-detects Next.js — no config needed
4. Deploy!

## Local Dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features
- Real-time sync via Socket.io
- 6 photo layouts (vertical, grid, strip)
- 3 border themes (pink, blue, mint)
- Camera with countdown & dual ready-check
- 10 photo filters
- Stickers, custom text, date stamp
- Export PNG / JPG / High-res print
- Unique session rooms with shareable links
